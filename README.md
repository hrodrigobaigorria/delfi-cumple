# 🎂 Delfi Cumple

Tarjeta de cumpleaños interactiva hecha con **React + TypeScript + Vite**.

## Requisitos

- Node.js 18 o superior

## Comandos

```bash
npm install      # instalar dependencias
npm run dev      # servidor de desarrollo (http://localhost:5173)
npm run build    # compilar para producción (genera /dist)
npm run preview  # previsualizar el build de producción
```

## Música 🎵

El botón "Tocar música" reproduce el archivo `public/delfi.mp3` en loop.
Para cambiar la canción, reemplazá ese archivo (o actualizá la ruta en `src/App.tsx`).
